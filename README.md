# Oasis Engine

A modern, multi-language platformer game engine with a unique architecture.

## Features

- **High Performance**: Rust backend for maximum performance.
- **Flexible Scripting**: Python for game logic and objects.
- **Modern UI**: Web-based user interface (HTML/JS).
- **Authoritative Server**: Secure server-client model.
- **Studio Tool**: Create and publish your own servers.

## Quick Start

### Prerequisites

- Rust (latest stable)
- Python 3.8+
- Node.js (for web assets)

### Running the Project

1. Clone the repository:
   ```bash
   git clone https://github.com/systech-x64/oasis.git
   cd oasis
   ```

2.Build and run the server:

   ```bash
   cd server
   cargo run
   ```

3.Build and run the client:

   ```bash
   cd client
   cargo run
   ```

4.Build and run the studio:

   ```bash
   cd studio
   cargo run
   ```
